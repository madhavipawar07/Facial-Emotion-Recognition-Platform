# Facial-Emotion-Recognition-Platform

## Overview

Detects facial emotions from uploaded images using Amazon Rekognition.

## Features

- Face Detection
- Emotion Detection
- Smile Detection
- Age Range
- Gender
- Confidence Score

## Technologies

- Python
- HTML
- CSS
- JavaScript

## AWS Services

- Rekognition
- Lambda
- S3
- DynamoDB
- API Gateway

## Workflow

Upload Image

↓

S3

↓

Lambda

↓

Rekognition

↓

DynamoDB

↓

Website
